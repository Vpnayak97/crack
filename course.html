About the Workshop

This hands-on IoT workshop helps students understand how connected devices communicate using sensors, microcontrollers, and cloud platforms. Learn to prototype real-world IoT systems, gather data, and build smart applications to bridge the gap between hardware and software innovation.

Workshop Syllabus

  • Introduction to IoT — sensors, actuators, and connectivity
  • Working with microcontrollers (NodeMCU/ESP32)
  • Data collection and cloud dashboards
  • Hands-on project: Smart Home prototype
  • Certification and career guidance

Schedule

Day 1: Basics, setup, and first project

Day 2: Advanced integration and final presentation
